# AI 工具使用说明

使用 ChatGPT/Codex 辅助：题面拆解、代码实现、像素级错误审计、评测脚本、论文与作者实现调研、AutoDL 实验编排、负结果整理和报告排版。关键的数据边界、正式方案取舍、C 语义区域判断、许可证风险与失败结论由人工复核负责。

本项目历经多轮长对话与实验，无法从界面导出精确 token 账单；按上下文规模保守估计约 20-40 万 token。消耗最多的是 B 大位移/遮挡反证，其次是 A 三折消融与 C 逐区域错色/白点审计。

使用一张 AutoDL RTX 4090 完成 A 预训练、LOFO 适配及 B 清理/拓扑消融。GPU 计费金额与完整累计时长未可靠记录，因此不编造成本数字；已知单次 A 5,000 步预训练约 9 分钟。未调用付费图像生成 API。
