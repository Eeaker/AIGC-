# 方法摘要

## A 描原

正式输出遵守题面四种精确色：白、黑、蓝、红。几何分支采用外部 Deep Sketch Vectorization 风格的中心线/截断距离场预训练，再在 KTK_04 三张成对图上做三折 leave-one-frame-out 适配；每个留出帧的参考只在推理完成后评分。保守闭运算和端点方向门控修补不超过 5 px 的短缝。参考成品存在绿色生产线，故将绿色恢复作为补充五色版本，不覆盖正式四色结果。

## B 中割

把端点线稿转换成截断距离场，以双向 DIS 估计粗运动；沿端点/交点间有序笔画正则位移，再变形八连通像素图的边，避免普通 splat 的断点。A6-A9 大转头使用 A0009 单拓扑反向场，红/绿稀疏语义线独立估计。正式 blind 版本不加载由 A2-A5 成品训练的局部残差 U-Net；该权重仅用于 shot-adapted 诊断上限。律表时刻固定为 A2-A5 的 0.25/0.50/0.75/0.90 与 A7/A8 的 0.50/0.75。

更新的 topology_visibility_v2 标注仅使用 A1/A6/A9 构建。33 个 A6-A9 人工语义点做过全局 TPS、部件图形变和局部残差融合：部件图直接形变最佳仅 0.165/0.230，局部融合平均 2 px F1 仅 +0.00010。预算化 Assisted-10 最佳也只从 0.55303 到 0.55390，未过预注册的 +0.002 多指标门槛；Assisted-30 未胜出。871 个 visibility 事件仍待人工复核，因此不宣称完整 GT 或通用可见性模型。

## C 上色

从设定图精确统计调色盘；对每帧线稿做四连通白区分解。正式 153 个标签描述四个关键视图 A1/A5/A7/A9 的 paint-bucket 区域语义，颜色来自线稿、角色设定图和色卡，不使用 A2-A8 中间成品误差修补。区域 mask 通过距离场运动和 inclusion 投票传播；冲突时保守留白。输出从输入副本开始，只修改原白像素，因此线稿保持严格为 0 变化。成品对照后扩充的 225 点版本仅作诊断。
