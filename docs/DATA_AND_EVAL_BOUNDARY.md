## 指标口径

- A/B 将所有非白像素作为联合线集，报告 exact、1 px、2 px 双向容差 F1；同时逐色计算，避免联合分数掩盖语义错色。
- A 额外报告严格四色符合率和连通域闭合泄漏。
- B 诊断实验还比较线像素比例和连通组件；F1 提升但线量显著膨胀时不接受。
- C 只在输入线稿的四连通白区内评测：面积精度衡量“涂出的是否正确”，正确覆盖衡量“参考目标中有多少被正确涂出”。
- C 必须单独断言所有输入非白线像素不变，不能用“视觉上相似”替代逐像素检查。
- 所有主均值均为逐帧宏平均，非将九帧像素合并后的池化结果。

## 标注状态

`annotations/task_b_topology/validation_report.json` 只证明数据结构合法，不等价于人工语义正确。当前 v2 中共有 33 个 A6--A9 人工语义点、24 条边可见性记录，另有 871 个 visibility 事件待复核。因此该包适合稀疏消融，不足以监督或宣称通用图匹配模型。
